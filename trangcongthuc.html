Công Thức Lớp 10

1. Phương trình bậc hai: ax² + bx + c = 0

Nghiệm: x = [-b ± √(b² - 4ac)] / (2a)

Điều kiện: a ≠ 0, Δ = b² - 4ac ≥ 0

2. Hệ phương trình tuyến tính 2 ẩn: a1x + b1y = c1, a2x + b2y = c2

Giải bằng phương pháp thế hoặc cộng đại số.

Nghiệm: x = (c1b2 - c2b1)/(a1b2 - a2b1), y = (a1c2 - a2c1)/(a1b2 - a2b1)

3. Công thức lượng giác cơ bản

sin²α + cos²α = 1

tanα = sinα / cosα

cotα = cosα / sinα

4. Diện tích tam giác: S = (1/2)ab sinC

Trong đó a, b là hai cạnh, C là góc giữa chúng.

5. Tập xác định hàm số: f(x) = √(x - a)

Tập xác định: x ≥ a

6. Bất đẳng thức Cauchy: (a + b)² ≥ 4ab

Dấu bằng khi a = b.

7. Công thức Heron: S = √[p(p-a)(p-b)(p-c)]

Diện tích tam giác với ba cạnh a, b, c; p = (a+b+c)/2

8. Phương trình đường thẳng: y = ax + b

a là hệ số góc, b là tung độ gốc.

Công Thức Lớp 11

1. Phương trình lượng giác: sinx = a

Nghiệm: x = arcsin a + kπ, x = π - arcsin a + kπ

Với |a| ≤ 1

2. Logarit: log_a b = c ⇔ a^c = b

Điều kiện: a > 0, a ≠ 1, b > 0

3. Giới hạn cơ bản: lim (x→0) (sinx / x) = 1

Lim (x→0) (1 - cosx)/x = 0

4. Đạo hàm: d/dx (x^n) = n x^{n-1}

d/dx (sinx) = cosx

d/dx (cosx) = -sinx

5. Phương trình đường thẳng trong không gian

Vector chỉ phương: u = (a, b, c)

Điểm M0(x0,y0,z0), phương trình: (x-x0)/a = (y-y0)/b = (z-z0)/c

6. Diện tích hình chiếu

S = S' cosα, với α là góc giữa mặt phẳng và đường thẳng vuông góc.

7. Công thức tích phân: ∫ x^n dx = x^{n+1}/(n+1) + C

∫ sinx dx = -cosx + C

∫ cosx dx = sinx + C

8. Bất đẳng thức lượng giác: sinx ≤ x (rad)

Cho x ∈ [0, π/2]

Công Thức Lớp 12

1. Ma trận nghịch đảo: A^{-1} = (1/det A) adj A

Điều kiện: det A ≠ 0

2. Tích phân xác định: ∫_a^b f(x) dx = F(b) - F(a)

F là nguyên hàm của f

3. Phương trình vi phân: dy/dx = f(x)

Nghiệm: y = ∫ f(x) dx + C

4. Xác suất: P(A) = số kết quả thuận lợi / tổng số kết quả

Định lý Bayes: P(A|B) = P(B|A) P(A) / P(B)

5. Phân phối chuẩn: Z = (X - μ)/σ ~ N(0,1)

P(Z < z) từ bảng chuẩn.

6. Công thức tích phân từng phần: ∫ u dv = u v - ∫ v du

Áp dụng cho tích phân phức tạp.

7. Giá trị cực đại/tiểu: f'(x) = 0, f''(x) > 0 (cực tiểu)

Kiểm tra đạo hàm bậc hai.

8. Thể tích khối tròn xoay: V = π ∫_a^b [f(x)]^2 dx

Quanh trục Ox.
